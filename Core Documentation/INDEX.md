## üö® COMMON ISSUES & SOLUTIONS

| Issue | Solution |
|-------|----------|
| Supabase connection error | Check `.env.local` has all 3 keys |
| User can't see profile data | Run `supabase-policies.sql` again |
| File upload fails | Verify bucket exists, check RLS policies |
| Typography doesn't match | Use exact pixel values, not rem/em |
| TypeScript errors | Run `npx supabase gen types typescript` |
